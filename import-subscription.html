📥 Clash Verge Rev 导入订阅指南

只需复制粘贴你的 Clash 订阅链接，一分钟内完成节点配置。支持多订阅管理、自动更新与自定义策略组。

1

获取订阅链接

从你的机场网站复制 Clash 订阅地址，或使用订阅转换工具生成。

2

打开配置页面

启动 Clash Verge Rev，点击左侧菜单的“配置”选项卡。

3

从 URL 导入

点击“新建” → “从 URL 导入”，粘贴链接并保存。

4

启用并验证

在配置列表中选中该配置，开启系统代理或 TUN 模式，测试网络连通性。

📌 三种导入方法详解

🔗 直接 URL 导入（推荐）

大多数机场会提供形如 https://your-airport.com/.../clash 的订阅链接。在 Clash Verge Rev 的“配置”页面点击“新建”，选择“从 URL 导入”，粘贴该链接并点击确定。客户端会自动下载并解析节点。

💡 提示： 如果链接在国内无法直接访问，可尝试使用订阅转换服务（见下文）。

📋 手动粘贴配置内容

如果你已通过其他方式获得了完整的 YAML 配置文件内容（例如从订阅转换工具下载），可在“配置”页面选择“新建” → “手动编辑”，将内容粘贴到文本编辑器中并保存。

📁 从本地文件导入

如果你已经将配置文件下载为 .yaml 或 .yml 文件，点击“从文件导入”选择该文件即可。适用于备份恢复或多设备同步。

🔄 订阅转换：让任意链接兼容 Clash

部分机场仅提供 SS/SSR/V2Ray 原始链接，此时需要借助订阅转换工具生成 Clash 可用的配置。常用的公共转换服务（请自行搜索稳定性较好的后端）接受原始订阅 URL 并输出 Clash 格式。

典型用法：https://转换后端地址?target=clash&url=你的原始订阅链接

⚠️ 隐私提醒： 使用第三方转换服务时，你的订阅链接可能会被服务端记录。建议使用自建转换后端或信任的服务。

⏳ 订阅管理技巧

🔄 自动更新订阅

在配置列表中，右键点击某个订阅配置，选择“编辑”，在“更新间隔”中输入时间（单位：分钟）。例如设置 1440 表示每天自动更新一次。更新后节点会自动同步。

📂 多订阅共存

你可以导入多个订阅配置（例如不同机场或不同策略组），在“配置”页面随时切换。适合同时拥有多个服务提供商的用户。

💾 手动强制更新

如果感觉节点延迟变高或部分节点失效，可以右键点击配置选择“更新”立即拉取最新节点列表，无需重新导入。

🔒 安全警示： 订阅链接是你个人专用凭证，请勿分享给他人或在不受信任的设备上使用。建议定期更换订阅密码，防止流量盗用。

❓ 导入订阅常见问题

为什么导入后没有节点？

请检查订阅链接是否有效，尝试在浏览器中打开该链接看是否返回 YAML 内容。部分机场需要后端转换或添加参数。

订阅链接提示“无效”怎么办？

确认链接以 http:// 或 https:// 开头，且没有多余空格。如果链接由机场提供但无法使用，联系机场客服获取 Clash 专用订阅地址。

更新订阅后节点丢失？

可能是机场的配置格式发生了变化。可尝试删除配置后重新导入，或检查机场公告。

能否导入多个订阅？

完全可以。每个订阅会作为一个独立的配置出现在列表中，你可以根据需要随时切换。
