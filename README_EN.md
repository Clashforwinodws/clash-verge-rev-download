This repository **does not provide any proxy subscriptions**. Here’s how to get and import a config:

1. **Obtain a subscription**  
   Get a subscription URL (usually a link or QR code) from the service provider of your choice.

2. **Import into Clash Verge Rev**  
   Open Clash Verge Rev → Go to **Profiles** → Paste your subscription link under **Import from URL** → Save and enable.

3. **Subscription conversion (optional)**  
   If your subscription isn’t directly compatible, use a public tool like `subconverter` to transform it into a Clash-friendly format.

   Advanced users can also import a local `yaml` config file to fully customize rules.
