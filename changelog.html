v2.5.2 2026-06-29 · 最新稳定版

新增 新特性

  • 新 支持 Hysteria2 协议，大幅提升高延迟网络下的传输效率。
  • 新 新增“连接状态”面板，可实时查看每条流的代理状态和延迟。
  • 新 引入配置快照功能，一键回滚到之前的配置版本。

修复 Bug 修复

  • 修 修复 TUN 模式下 DNS 解析偶发超时的问题。
  • 修 修复导入大量节点时 UI 卡顿的现象。
  • 修 修复 Linux 下托盘图标在 KDE 环境中的显示异常。

改进 性能优化

  • 优 优化 Mihomo 内核的 UDP 转发性能，游戏延迟降低约 15%。
  • 优 降低 Fake‑IP 模式下的内存占用，大规模规则集场景更流畅。
v2.5.1 2026-05-14

新增

  • 新 支持 Rule Provider 热更新，无需重启即可刷新规则集。
  • 新 新增“网络唤醒”功能，允许局域网设备通过 WoL 唤醒。

修复

  • 修 修复在 macOS Ventura 上窗口无法置顶的问题。
  • 修 修复某些机场订阅解析失败的错误。
v2.5.0 2026-04-02

新增

  • 新 完全重构 UI，采用 Tauri 2.0 框架，启动速度提升 40%。
  • 新 新增暗色模式及多种主题配色。
  • 新 支持 WireGuard 协议，可作为轻量级 VPN 客户端。

改进

  • 优 重新设计设置页面，分类更清晰，搜索更方便。
v2.4.0 2025-12-10

新增

  • 新 实验性支持 QUIC 协议，降低握手延迟。
  • 新 新增“规则调试”工具，可测试特定 URL 的匹配结果。

修复

  • 修 修复部分 Linux 发行版上 TUN 模式无法自动获取 DNS 的问题。
