🛠️ Clash Verge Rev 安装教程

无论你使用 Windows、macOS 还是 Linux，只需跟随这份分步指南，即可在几分钟内完成客户端的安装与基础部署。

📋 系统要求

🪟 Windows

  • Windows 10 或更高
  • x64 或 ARM64 处理器
  • 4 GB 内存
  • 200 MB 磁盘空间

🍎 macOS

  • macOS 10.15 及以上
  • Intel 或 Apple Silicon
  • 4 GB 内存
  • 250 MB 磁盘空间

🐧 Linux

  • Ubuntu 20.04+ / Debian 11+
  • Fedora 36+ / Arch 最新
  • 4 GB 内存
  • AppImage 或原生包

📱 移动端

  • Android 7.0+
  • iOS 14.0+ (需自签或 TestFlight)
  • 至少 100 MB 空间
  • 网络连接

📥 各平台安装步骤

🪟 Windows 安装指南
  1. 前往 下载页面 或直接访问 GitHub Releases。
  2. 找到最新版本，下载 Clash.Verge_2.5.2_x64-setup.exe (64位) 或对应的 ARM64 版本。
  3. 双击安装文件，如果出现 SmartScreen 警告，点击“更多信息” → “仍要运行”。
  4. 按照安装向导操作，建议勾选“创建桌面快捷方式”。
  5. 完成安装后，从开始菜单或桌面启动 Clash Verge Rev。
  6. 首次启动可能提示防火墙权限，请允许“专用网络”访问。
  7. 此时客户端已成功安装，下一步请参考 配置教程 导入节点。
🍎 macOS 安装指南
  1. 从 下载中心 获取 Clash.Verge_2.5.2_aarch64.dmg (Apple Silicon) 或 Intel 版本。
  2. 打开 .dmg 文件，将 Clash Verge Rev 拖拽到“应用程序”文件夹。
  3. 由于应用未经过 Apple 公证，首次打开时可能提示“无法验证”。前往“系统设置” → “隐私与安全性” → 点击“仍要打开”。
  4. 或者直接在 Finder 中右键点击应用，选择“打开”，确认对话框后即可启动。
  5. 启动后，菜单栏会出现 Clash 图标，表示已成功运行。
🐧 Linux 安装指南
  1. 下载适用于你发行版的安装包：.deb、.rpm 或通用 .AppImage。
  2. Debian/Ubuntu： 双击 .deb 文件或在终端执行 sudo dpkg -i ./Clash.Verge*.deb。
  3. Fedora： 执行 sudo rpm -i ./Clash.Verge*.rpm。
  4. AppImage： 赋予执行权限 chmod +x ./Clash.Verge*.AppImage，然后双击或 ./Clash.Verge*.AppImage 运行。
  5. 安装后，可从应用菜单或命令行启动。
📱 移动端安装 (Android / iOS)

Android：

  1. 从 GitHub Releases 下载 .apk 文件。
  2. 在设置中允许“安装未知应用”，然后点击 APK 安装。
  3. 启动后授予必要的 VPN 权限。

iOS：

  1. 由于限制，iOS 版本通常通过 TestFlight 分发或自签 IPA 安装。
  2. 请关注社区公告获取 TestFlight 邀请链接。
  3. 安装后需在“设置” → “通用” → “VPN 与设备管理”中信任证书。

✅ 安装完成？验证一下

启动客户端后，你会在系统托盘（Windows右下角/ macOS菜单栏）看到 Clash 图标。右键点击图标，确认“系统代理”或“TUN 模式”可以正常开启。如果图标显示为灰色，请检查是否已经导入有效的节点配置。

❓ 安装常见问题

Windows 提示“已阻止此应用”

这是因为应用未使用微软商店签名。点击“更多信息”，然后选择“仍要运行”即可正常安装。

macOS 无法打开，提示来自身份不明的开发者

前往“系统设置” → “隐私与安全性”，在底部找到“已阻止使用…”，点击“仍要打开”。或者右键点击应用图标选择“打开”。

Linux 下 AppImage 无法运行

确保已安装 FUSE 库：sudo apt install libfuse2 (Ubuntu) 并检查执行权限。

安装后启动没有反应

尝试从终端启动以查看错误日志。Windows 可查看事件查看器，Linux/macOS 在终端输入应用名运行。
