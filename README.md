# Clash Verge
## 什么是 Clash Verge？

Clash Verge 是一款基于 Clash.Meta（Mihomo）核心开发的现代化代理客户端，提供简洁美观的用户界面以及强大的网络代理功能。它支持 Windows、macOS 和 Linux 平台，可帮助用户轻松实现规则分流、节点管理、订阅更新以及系统代理配置。

相比传统 Clash 客户端，Clash Verge 拥有更加现代化的 UI 设计、更优秀的性能表现以及更加丰富的功能扩展，是目前广受欢迎的 Clash GUI 客户端之一。

## Clash Verge 功能特色
### 现代化界面设计

Clash Verge 使用全新的界面设计语言：

1. 简洁直观
2. 深色模式支持
3. 实时日志显示
4. 节点延迟测速
5. 配置管理更方便

即使是新手用户也可以快速上手。

### 支持多平台代理

Clash Verge 支持：Windows、macOS、Linux

用户可在不同系统之间保持一致的代理体验。

### 支持多种代理协议

Clash Verge 基于 Clash Meta 内核，支持：

1. Shadowsocks（SS）
2. ShadowsocksR（SSR）
3. Vmess
4. VLESS
5. Trojan
6. Snell
7. SOCKS5
8. HTTP
9. Hysteria
10. TUIC

兼容绝大多数主流机场订阅。
